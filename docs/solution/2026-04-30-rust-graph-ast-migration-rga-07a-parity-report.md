## Deletion gate assessment

RGA-08 remains blocked.

RGA-07A-specific blockers:

1. No existing full official-corpus TypeScript baseline generator was found in package scripts.
2. The only available TypeScript baseline path is legacy TS graph tooling, and it covered only 35 files in this corpus.
3. The Rust parity CLI consumed only root-level fixture files from the supplied workspace and produced a degraded one-case result.
4. Normalized parity is count-level and common-file bounded; it does not provide identity-level symbol/import/call/reference equivalence.
5. Required thresholds are therefore not proven:
   - symbols ≥ 99%;
   - imports/dependencies including cross-root ≥ 99%;
   - calls/references ≥ 95% with gaps triaged.

RGA-07B/RGA-07C blockers from the broader deletion gate remain out of scope for this split task and still need separate evidence: payload, event-loop, memory, rollback rehearsal, and mutation-based incremental gates.
